开发标注版

Design Handoff v1.0

本页用于前端开发对齐，包含尺寸、间距、色值、字体、组件命名和页面映射。所有数值与 `proto/assets/theme.css` 和各页面原型保持一致。

1. 设备与布局标注

屏幕规格（原型基线）

画板：390 x 844

外壳圆角：34

状态栏高度：34

底部导航高度：68（距离底部 12）

内容区内边距：16（上/左/右）+ 96（底）

间距基准（8pt 体系）

XS: 4 / S: 8 / M: 12 / L: 16 / XL: 24 / 2XL: 32

卡片圆角：16-18

输入框高度：44

按钮高度：44（主按钮）

模块间距：12-16

2. 颜色与字体标注

主色 Token

--pink-500: #ff7fae（强调按钮/高亮）

--blue-500: #5da8ff（信息/可操作提示）

--yellow-300: #ffe186（提醒/标签）

--line: #ffd5e7（边框）

--ink: #3f3b4f（主文本）

字体 Token

标题字：ZCOOL KuaiLe（.title-font）

正文字：Noto Sans SC / Nunito

H1: 30 / 700

Section: 15 / 800

正文: 13-14 / 400-500

辅助文案: 11-12

4. 动效与状态标注

页面进入动效

`bounce-in`: 520ms / ease-out / translateY(12px) + scale(0.98 → 1)

临近拍摄提醒

`pulse-alert`: 1.5s 无限循环，用于首页紧急卡片闪烁提醒。

按钮按压状态

建议补充 `:active { transform: scale(0.98); filter: brightness(.96); }`。

冲突状态

排单冲突统一使用黄色模块 + 警告图标 + 冲突对象与时间。
